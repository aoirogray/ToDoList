# FlowTodo (タスク＆時間管理Webアプリ)

FlowTodo は、スマートフォンでの利用を中心に設計された、シンプルかつ洗練されたデザインのタスク・時間管理用シングルページWebアプリケーション（SPA）です。

## 🌟 主な特徴

- **モバイルファースト設計**: スマートフォンでの操作性を最優先し、親指で操作しやすいボトムナビゲーションバーや、スライドイン式の入力フォーム（ボトムシート）を搭載。PC表示時もスマートフォンサイズに中央配置され、スタイリッシュな外観を維持します。
- **プレミアムダークテーマ & グラスモーフィズム**: 深みのあるネイビーを基調に、半透明のガラス効果（Backdrop-filter）と滑らかなグラデーション、アニメーションを採用したモダンなUI。
- **ワンタップ完了**: チェックボックスを1回タップ（クリック）するだけで、心地よいスケール＆フェードアウトアニメーションとともに「完了グループ」へタスクが移動します。
- **タスク＆時間管理機能**: 
  - タスク登録（タイトル、優先度、見積もり時間（時間・分）、期限日時、メモ書き）
  - タスクの編集・削除
  - 未完了（Active）と完了（Completed）の自動グループ化表示（完了グループはアコーディオン式に折りたたみ可能）
- **カスタムカテゴリ機能**: 
  - デフォルトカテゴリ（仕事、プライベート、買い物、健康、未分類）に加え、ユーザー自身が好きなカテゴリを自由に追加・削除できます。
  - 各カテゴリにはテーマカラーを設定可能で、タスクカードに色付きバッジとして反映されます。
- **統計・進捗のビジュアル表示（ダッシュボード＆分析）**:
  - **ホーム**: 今日の進捗率（SVG円形進行ゲージ）、未完了・完了数、見積もり時間のサマリー、優先度の高い「高」タスクの抽出表示。
  - **分析**: `Chart.js`を使用した「タスク完了率（ドーナツチャート）」と「カテゴリ別の見積もり時間配分（横棒グラフ）」による時間の見える化。
- **データローカル保存**: 全データがブラウザの `localStorage` に自動保存されるため、サーバー設定不要でリロードしてもデータは保持されます。

## 🚀 起動方法

### 方法 1: ブラウザで直接開く（最も簡単）
`index.html` ファイルをブラウザ（Google Chrome, Safari, Edgeなど）にドラッグ＆ドロップするか、ダブルクリックするだけで即座に起動します。追加のインストールやセットアップは一切不要です。

### 方法 2: ローカルWebサーバー経由で開く（推奨）
PCにPython、PHP、またはVS Codeがインストールされている場合は、ローカルサーバーを起動してアクセスすると、より安定して動作します。

- **Python がインストールされている場合**:
  ターミナル（PowerShellやCommand Prompt）を開き、本ディレクトリで以下のコマンドを実行します：
  ```bash
  python -m http.server 8000
  ```
  実行後、ブラウザで `http://localhost:8000` にアクセスしてください。

- **PHP がインストールされている場合**:
  ```bash
  php -S localhost:8000
  ```
  実行後、ブラウザで `http://localhost:8000` にアクセスしてください。
